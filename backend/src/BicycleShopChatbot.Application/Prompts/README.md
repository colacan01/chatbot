# Prompt Templates

이 디렉토리는 챗봇의 시스템 프롬프트 템플릿 파일들을 포함합니다.

## 파일 목록

- **ProductSearch.txt** - 제품 검색/추천 시 사용되는 프롬프트
- **ProductDetails.txt** - 제품 상세 정보 조회 시 사용되는 프롬프트
- **FAQ.txt** - FAQ 질문 응답 시 사용되는 프롬프트
- **OrderStatus.txt** - 주문 조회 시 사용되는 프롬프트
- **CustomerSupport.txt** - 고객 지원 시 사용되는 프롬프트
- **General.txt** - 일반 대화 시 사용되는 프롬프트

## 사용 방법

1. 각 텍스트 파일을 직접 편집하여 프롬프트를 수정할 수 있습니다.
2. 파일을 수정한 후 애플리케이션을 재시작하면 변경사항이 적용됩니다.
3. 파일이 존재하지 않거나 읽을 수 없는 경우, PromptService에 정의된 fallback 프롬프트가 사용됩니다.

## 주의사항

- 파일 인코딩은 UTF-8을 사용해야 합니다.
- 프롬프트 내용을 수정할 때는 AI의 역할과 제약사항을 명확히 유지하세요.
- 프로덕션 환경에서는 프롬프트 변경 시 충분한 테스트를 거쳐야 합니다.
